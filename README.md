# School Network Design in Cisco Packet Tracer

![Network Topology](topology.png)

## Overview
This project is a small school network designed in Cisco Packet Tracer. The network is divided into separate segments for the server room, IT department, faculty/staff, students, and printer access. The goal was to create a network that is organized, secure, and efficient for a school environment.

## Features
- Network segmentation by department
- Subnetting for better IP address organization
- Static routing between routers
- Access Control List (ACL) to restrict access to sensitive network areas
- Separate public and private printer access

## Network Purpose
The design focuses on controlling access to important resources. The IT department has broader access across the network, while access to sensitive systems such as the server room is restricted. This helps improve security, reduce misuse of shared resources, and support the needs of students, staff, and administrators.

## Skills Demonstrated
- Cisco Packet Tracer network design
- IP addressing and subnetting
- Static route configuration
- Basic network security with ACLs
- Network documentation
